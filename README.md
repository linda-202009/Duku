# Duku运维操作平台

Duku提供基本的CMDB管理,SQL审核,SQL优化,用户权限管理等功能

环境需求:Django 2.2, AdminLte3, Bootstrap4, GoInception, Soar

进度列表:Soar、权限管理前端未完成,公有云API未集成,在开发中.....
